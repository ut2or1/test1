# K64 Blinky

Простой проект мигания светодиодом для микроконтроллера NXP Kinetis K64FX512 (FRDM-K64F).

## Возможности

- Мигание красного светодиода (PTC21) с частотой ~1 Гц
- Bare-metal реализация без HAL/ SDK
- Собственный стартап-код и таблица векторов

## Требования

- ARM GCC Toolchain (`arm-none-eabi-gcc`)
- Make

## Сборка

```bash
make
